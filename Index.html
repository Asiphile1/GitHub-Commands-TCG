git clone

Creates a copy of an existing Git repository.

Syntax:
git clone <repository-url>
Example Usage:
git clone https://github.com/user/repo.git
Explanation:

This command downloads the repository from the specified URL to your local machine, creating a local copy you can work on.

git push

Uploads local repository content to a remote repository.

Syntax:
git push <remote> <branch>
Example Usage:
git push origin main
Explanation:

This command sends your committed changes from your local repository to the remote repository, making them available to others.

git pull

Fetches and merges changes from a remote repository to your local branch.

Syntax:
git pull <remote> <branch>
Example Usage:
git pull origin main
Explanation:

This command updates your local repository with changes from the remote repository, combining them with your local work.

git commit

Records changes to the repository with a descriptive message.

Syntax:
git commit -m "commit message"
Example Usage:
git commit -m "Add new feature"
Explanation:

This command saves your staged changes in the local repository along with a message describing the changes.
